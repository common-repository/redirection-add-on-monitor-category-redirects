== Description ==

Monitor category redirects is an add-on for the Redirection plugin that allows you to monitor category redirects.

Whenever a category is updated, the plugin will automatically create a redirect from the old category URL to the new one.

== Installation ==

1. Install and activate the Redirection plugin.
2. Install and activate the Monitor category redirects add-on.

== Frequently Asked Questions ==

= How do I know if the add-on is working? =

If you update a category, you should see a new redirect in the Redirection plugin.

= How reliable is this add-on? =

The add-on is reliable, but it's not perfect. It's possible that the add-on will not work in some cases. When this happens, errors will be logged in the PHP error log.
